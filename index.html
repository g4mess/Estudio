code . :se utiliza par abrir archivos en VScode

Comandos para git y Utilizacion

git add . :actualiza todos los archivos modificados en el pc
git commit -m: "nombre de que se actualizo": para actualizar y subir a github
git push -u origin main: Otorgar permisos y terminar de subir archivos a github
git init: para colocar un repositoria de la carpeta
git status: para mirar que estado tiene los archivos en el limbo de git
git rm --cached [nombre]: eliminar archivos que estan en el limbo
git config: para mirar la configuracion de git
git log [nombre]: Para mirar todos los cambios que ha sufrido el archivo
git show [nombre]: Para mirar los cambios de los 2 ultimos cambios
gif diff [Version-a-vieja] [Version-b-Nueva]: Para comparar cambios de un archivo
git reset head~1 o version :esto es para resetear los ultimos cambios de una version
git log --stat: Muestra todos los cambios especificos de las versiones
git checkout [version]: se mueve de versiones de una a otra
git revert [version]: revierte los ultimos cambios de una version
git clone [url]: trae archivos de afuera donde esta el save total
git commit -am "mensaje": hace un commit de archivos viejos sin necesidad de add
git branch "nombre de la rama" : Para crear una nueva ramificacion
git merge [version]: Para combinar la version y agregar una rama
git pull origin main: Para traer cosas que no tenemos de github a tu entorno
git pull origin main --allow-unrelated-histories: Fuerza el cambio sin acomodar las historias
git remote set-url origin [https o ssh]: cambia el vinculo donde esta conectado el git
