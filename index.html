Estructura de un sitio web

Hmtl: Es la esctructura central como el esqueleto que tiene el pagina
CSS: Luego aparece Css para darle color y estilo a la estructura
JS: el javascript queda conformado internamente de la maquina donde podemos realizar multiples interacciones con codigo

De esta estructura nacen las siguientes arboles de organizacion:
DOM: la conecion de arbol qu8e muestra toda la estructura y el js precarcado
CSSOM: la estructura de diseño que se precarga para saber donde tiene que ir ubicada
Render Three: es el arbol al unirse DOM y CSS
Layout: es la ubicacione que va a tener cada uno de los objetos de la pagina web
Paint: Es el entorno donde organiza todo para la realizacion del compilado final.
